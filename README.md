## 🛠️ How to Set Up the Website in Your Local Environment

To run this website on your own computer, follow the steps below:

---

### 📦 Step 1: Download the Project Files

You need to download both **client-side** and **server-side** source code:

- 🔽 **Client Side:** [Download or Clone from GitHub](https://github.com/your-username/client-repo)
- 🔽 **Server Side:** [Download or Clone from GitHub](https://github.com/your-username/server-repo)

---

### 🎨 Step 2: Configure the Client Side

After downloading, go to the client-side folder and follow these steps:

1. Open the project in your code editor.
2. Run `npm install` to install dependencies.
3. Create a `.env` file and set the following values:
   ```env
   VITE_API_URL=http://localhost:5000
   VITE_PUBLIC_KEY=your_public_key_here
